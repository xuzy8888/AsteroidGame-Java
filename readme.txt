Instruction:
run the Game.java file in the mvc/controller folder
Press s to start
Press left/right arrow key to move to left or right
Press space to fire

Each level there will be opponents aliens generated that will drop bomb.
Bullets and bombs can not penetrate the shelters (the 2 white rectangles in the middle
of the game panel).
Clearing each wave of enemies will gain 100 points.
At some level there will be Boss aliens generated, each of those need to be destroyed
with more than 1 bullets hits, and they can drop more powerful bombs that can penetrate
the shelters.

Notice:
Sometimes the game will throw currentModificationException and all enemies will stop
fire. Since it doesn't show up everytime, and I haven't found out a way to fully
eliminate this, if it happens please restart the game for now.
